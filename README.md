# Março Amarelo e Lilás

Projeto de página web desenvolvido para divulgar e conscientizar sobre as campanhas **Março Amarelo** e **Março Lilás**, que abordam temas importantes relacionados à saúde da mulher.

## Objetivo

O objetivo do projeto é apresentar informações de forma clara e acessível sobre as campanhas de conscientização realizadas no mês de março, incentivando a prevenção e o diagnóstico precoce de doenças.

## Tecnologias utilizadas

- HTML5
- CSS3


## Como visualizar o projeto

1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
